# 🚢 Titanic Survival Prediction

## 📖 Overview
This project predicts whether a passenger survived the Titanic disaster using passenger information and machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🔄 Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Model Training
- Model Evaluation

## 📊 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## ✅ Result
The classification model successfully predicted passenger survival with good accuracy.
