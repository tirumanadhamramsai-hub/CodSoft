# 🌸 Iris Flower Classification

## 📌 Project Overview
This project builds a Machine Learning classification model to predict the species of an Iris flower based on its sepal and petal measurements.

## 🎯 Objective
To classify Iris flowers into one of three species using machine learning classification algorithms.

## 📂 Dataset
The dataset contains 150 flower samples with the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Target)

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Workflow
- Data Loading
- Data Exploration
- Data Visualization (EDA)
- Label Encoding
- Train-Test Split
- Random Forest Classification
- Model Evaluation

## 📈 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## ✅ Result
The Random Forest Classifier achieved excellent accuracy in classifying Iris flower species.
